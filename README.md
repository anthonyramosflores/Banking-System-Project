# Banking System Project
A java-based banking application using objects for programming II class

## Features
- creates a BankAccount and adds it to the accounts list if and only if the account number is unique AND the initial deposit is not negative
- transfers a positive sum of money between two different accounts if and only if the fromAccount contains enough money for the transfer
-  prints the account information if and only if an account with the given number exists, or throws exception if not found
 - returns sum of all account balances
